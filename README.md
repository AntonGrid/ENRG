# ENRG Protocol

**The Backbone for DePIN Energy**

ENRG is a DePIN-native energy protocol that turns real-world energy production into a programmable, on-chain asset. Each ENRG token represents **1 MWh** of verifiable energy, backed by cryptographic proofs from IoT devices and oracles.

## Key Features
- **Proof-of-Production** – IoT hardware signs energy data, verified on-chain
- **Fixed supply** – 1,000,000,000 ENRG (hard cap)
- **Buyback & Burn** – 20% of protocol fee buys back and burns tokens
- **Solana** – fast, low-cost transactions
- **Real MWh backing** – not loyalty points
