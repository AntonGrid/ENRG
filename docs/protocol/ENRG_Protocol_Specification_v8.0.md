## Overview

This document is the normative specification of the ENRG Protocol.

The specification is organized as independent chapters located in:

```text
docs/protocol/chapters/
```

Each chapter defines one logical aspect of the protocol.

Together they constitute the complete ENRG Protocol Specification.
