# ENRG Cryptography Model

## Статус
Черновик (Draft) v0.1

## Введение
Криптографическая модель ENRG построена на принципе **нулевого доверия к серверу**. Устройства самостоятельно генерируют и хранят приватные ключи, а сервер (оракул, реестр) использует только публичные ключи для верификации подписей.

---

## 1. Ключевые алгоритмы

### 1.1 Ed25519
Используется для подписи Proof и других сообщений устройства.

- **Генерация ключей:** на устройстве (ESP32) при первом запуске.
- **Приватный ключ:** никогда не покидает устройство.
- **Публичный ключ:** отправляется на сервер при регистрации.
