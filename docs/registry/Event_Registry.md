# ENRG Event Registry

## Status

Normative

---

# Overview

This registry defines the official protocol event names.

Only event names registered in this document SHALL be considered standard ENRG events.

---

# Device Events

| Event | Description |
|--------|-------------|
| device.registered | Device successfully registered |
| device.claimed | Device ownership claimed |
| device.provisioned | Device successfully provisioned |
| device.activated | Device entered ACTIVE state |
| device.quarantined | Device entered QUARANTINE state |
| device.maintenance | Device entered MAINTENANCE state |
| device.revoked | Device permanently revoked |

---

# Proof Events

| Event | Description |
|--------|-------------|

| proof.submitted | Proof submitted to Oracle |
| proof.verified | Proof successfully verified |
| proof.rejected | Proof rejected |
