# ENRG Specifications

## Status

Normative

---

## Overview

This directory contains normative documents that complement the ENRG Protocol Specification.

These documents define terminology, conformance requirements, and additional protocol specifications shared by all compliant ENRG implementations.

---

## Documents

### ENRG Terminology

Defines the official protocol terminology.

Every normative term used throughout the ENRG Protocol SHALL have exactly one official definition.

---

### ENRG Conformance

Defines the requirements that implementations SHALL satisfy in order to claim compliance with the ENRG Protocol.

---

## Relationship to the Protocol Specification

These documents are normative extensions of the ENRG Protocol Specification.

If a conflict exists between documents, the ENRG Protocol Specification SHALL take precedence unless explicitly stated otherwise.
